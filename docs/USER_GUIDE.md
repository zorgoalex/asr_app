# Voice ASR Client — краткая инструкция

## Запуск
Приложение работает в фоне и появляется в системном трее.

## Диктовка
- Нажмите горячую клавишу и удерживайте для режима `hold`.
- В режиме `toggle` нажмите один раз для старта и ещё раз для остановки.
- После распознавания текст вставится в активное окно.

## Меню в трее
- `Статус` — текущее состояние (Idle/Recording/Transcribing/Error).
- `Проверка соединения` — проверка доступа к Groq API.
- `Настройки` — параметры клиента.
- `Выход` — завершение работы.

## Настройки
- `Groq API key` — ключ доступа.
- `STT модель` — модель распознавания.
- `Язык` — `auto`, `ru`, `en`.
- `Hotkey` — строка вида `Ctrl+Alt+Space`.
- `Режим записи` — `hold` или `toggle`.
- `Микрофон` — устройство ввода.
- `Timeout (сек)` — таймаут запроса.
- `Лимит записи (сек)` — максимальная длительность записи.
- `Вставка текста` — `direct`, `clipboard`, `clipboard_only`.
- `Логирование` — `info` или `debug`.
- `Автозапуск с Windows` — запуск при входе в систему.

## Статусы
- `Idle` — ожидание.
- `Recording` — запись.
- `Transcribing` — распознавание.
- `Error` — ошибка.

## Ограничения
- Вставка текста может не работать в приложениях с повышенными правами.
- Без API‑ключа распознавание недоступно.
