# Как получить Groq API key

## Шаги
1. Зарегистрируйтесь в Groq Console.
2. Создайте новый API key.
3. Скопируйте ключ и сохраните в безопасном месте.
4. Откройте настройки приложения и вставьте ключ в поле `Groq API key`.

## Безопасность
- Не публикуйте ключ в открытом виде.
- Если ключ утёк, создайте новый и замените в настройках.

## Проверка
Используйте пункт меню `Проверка соединения` в трее.
