# Диагностика и типовые проблемы

## Где логи
- Логи: `%LocalAppData%\VoiceASRClient\logs\voice-asr-client.log`
- Конфиг: `%AppData%\VoiceASRClient\config.json`

## Чек‑лист
- Проверить API‑ключ в настройках.
- Проверить выбранный микрофон.
- Проверить доступ в интернет.
- Проверить статус в трее.

## Ошибки API
- `401/403` — неверный ключ или нет доступа.
- `429` — лимиты API.
- `5xx` — ошибка провайдера, повторить позже.

## Нет звука
- Убедитесь, что микрофон выбран и доступен.
- Закройте приложения, которые могут захватывать микрофон эксклюзивно.

## Hotkey не срабатывает
- Попробуйте другую комбинацию.
- Проверьте, не перехватывает ли hotkey другое приложение.

## Вставка текста не работает
- Для elevated приложений требуется запуск клиента с такими же правами.
- Используйте режим `clipboard` или `clipboard_only`.

## Проблемы с прокси
По умолчанию прокси не используется. Для включения:
```powershell
$env:VOICE_ASR_USE_PROXY="1"
```

## Сброс настроек
Удалите файл конфигурации:
`%AppData%\VoiceASRClient\config.json`
